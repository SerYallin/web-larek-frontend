# Просесы в проекте

## Главная страница.

 - Выбрать продукт.
 - Добавить продукт в корзину.
 - Открыть корзину.

## Корзина.

 - Удалить продукт из корзины.
 - Оформить заказ.

## Оформление заказа.

 - Выбрать способ оплаты.
 - Ввести адрес доставки.
 - Ввести адрес почты.
 - Ввести номер телефона.
 - Отправить заказ.
